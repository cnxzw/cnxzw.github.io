利用GitHub与Hexo构建个人博客

为了更好的总结自己的学习收获，本人决定从即日起养成写博客的习惯。首先要解决的就是如何开始自己的博客之旅，本着Geek的精神，在知乎上花了不到一天的时间，参考了吴润的GitHub+Hexo搭建个人网站详细教程，学习了怎么通过GitHub和Hexo框架来搭建自己的博客，下面就是对此方法的记录与总结。

关于Hexo


Hexo是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。网页可以托管在GitHub和Heroku上，我们选用的是GitHub。由于前期主要用于试验（其实就是穷），并没有购买域名，相关内容暂时不做说明。

搭建步骤

1.安装Git
2.安装Node.js
3.安装Hexo
4.推送网站
5.更换主题
6.发布文章

1.安装Git

Git是开源的分布式版本控制系统，用来高效处理项目。从Git官网 下载合适的安装包，然后进行安装，在命令行里输入git查看安装是否成功。成功安装后，鼠标右击运行Git Bash，设置user.name和user.email
